## 一、Maxwell 之前：四個獨立的定律

| # | 定律 | 發現者 | 年份 | 公式 |
|---|------|--------|------|------|
| 1 | 電場高斯定律 | Gauss | 1835 | ∇·E = ρ/ε₀ |
| 2 | 磁場高斯定律 | （已知） | — | ∇·B = 0 |
| 3 | 法拉第定律 | Faraday | 1831 | ∇×E = −∂B/∂t |
| 4 | 安培定律 | Ampère | 1823 | ∇×H = J |

**問題**：安培定律不自洽。取散度 → ∇·J = 0，但電荷守恆要求 ∇·J + ∂ρ/∂t = 0。

## 二、Maxwell 做了什麼（1865）

### 發現的 bug
安培定律少了一項。

### 修正
> **∇×H = J + ∂D/∂t**

加了位移電流 ∂D/∂t。一項。

### 從修正中得到的新東西

| # | 新結果 | 公式 | 性質 |
|---|--------|------|------|
| 1 | 位移電流 | ∂D/∂t | 修正（bug fix） |
| 2 | 光速可從常數算出 | c = 1/√(ε₀μ₀) | 新公式 |
| 3 | 光 = 電磁波 | 波動方程的解 | 新預測 |

### 驗證時間線
- 1865：Maxwell 發表統一理論
- 1887：Hertz 實驗驗證電磁波（22 年後）
- 1901：第一個諾貝爾物理獎頒給 Röntgen（X 射線 = 電磁波）
- Maxwell 本人 1879 年去世，未見到驗證

## 六、核心公式並排

### Maxwell 的統一
```
統一前：    ∇×H = J                    （Ampère，不完整）
統一後：    ∇×H = J + ∂D/∂t            （Maxwell，完整）
預測：      ∂²E/∂t² = c² ∇²E           （電磁波方程）
其中：      c = 1/√(ε₀μ₀)              （光速）
```

### Huang 的統一
```
統一前：    F² ≥ exp(−ΔD)               （JRSWW，純數學）
統一後：    τ = 1 − F ∈ [0,1]           （時間箭頭的度量）
           τ ≤ 1 − exp(−Σ/2)           （開放環境的代價）
等價鏈：    τ=0 ⟺ Σ=0 ⟺ QMC           （封閉 = 可逆 = Markov）

延伸到重力：
           Σ_grav = r_s/r              （重力 = 資訊損失）
           c_eff/c = exp(−Σ_grav/2)     （光速 = 恢復保真度）
           n(r) = exp(+Σ_grav/2)        （折射率 = 1/保真度）

延伸到坍塌：
           τ(t) = 1 − exp(−E_G t/2ℏ)   （坍塌是連續過程）
```
